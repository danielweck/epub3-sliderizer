E-readers

Slideshows made with EPUB3-Sliderizer have been tested with:

  • iOS (WebKit)
    • iBooks, Kobo
  • Mac OS X, Windows
    • Readium.js (cross-browser)
    • Readium (Chrome extension) (WebKit)
    • Readium-SDK (WebKit + Gecko/Trident?)
    • iBooks (WebKit) (OSX 10.9+ aka "Mavericks")
    • Azardi (Gecko)
    • Calibre (Webkit)

Not working:
Google Play Books (no Javascript, destroys original HTML + CSS), BlueFire (breaks layout) (many others untested ...)
