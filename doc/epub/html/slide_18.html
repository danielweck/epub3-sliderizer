Author mode (alpha)

  1. Press the a key to toggle author mode (click outside of the WYSIWYG area, if already editing text).
  2. Hit esc to toggle the Markdown / HTML "source code" editor. You can also edit directly in WYSIWYG mode (HTML slides only, not XHTML).
  3. Changes are saved automatically to the browser local storage when switching in and out of the "source code" editor. In WYSIWYG mode, press ctrl + s at any time to save changes (or with Apple cmd).
  4. Use ctrl + d (or Apple cmd) to discard changes. This reverts back to the original data.txt state.
  5. The window title and the background color let you know the "modified" status of a slide.
  6. To save your edits permanently, you must copy + paste the text from the "source code" editor into the master data.txt.
  7. The images and text below can be moved using the mouse. To move the editable text below, use the ctrl modifier key (or Apple cmd).
