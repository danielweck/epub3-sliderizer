Author mode (experimental)

  1. Press the a key to toggle "author" mode. The slide will expand horizontally, so shrink the window if you need to.
  2. Once in "author" mode, press escape to switch to the Markdown editor. Note that changes are discarded when the page gets reloaded. To save your modifications, you must copy + paste into the master data.txt.
  3. The images and text below can be moved relative to their original position, using the mouse. Double-click on an image to see the CSS style (select + copy the text into the clipboard).
