Author mode (alpha)

  1. Press the a key to toggle author mode (click outside of the text-edit area if necessary).
  2. Press esc to switch to the Markdown / HTML editor, or to commit the live changes made on the WYSIWYG page.
  3. Hit the d key to discard your changes, which are saved into the browser local storage (this reverts back to the original data.txt state).
  4. To save your modifications permanently, copy + paste from the editor into the master data.txt.
  5. The images and text below can be moved using the mouse. To move text, use the ctrl modifier key (or Apple cmd).
